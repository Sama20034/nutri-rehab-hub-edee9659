

# تحسين عرض بيانات الطلبات + إشعارات البريد الإلكتروني

## المشكلة
1. **بيانات الطلب ناقصة**: الجدول بيعرض بس رقم الطلب والتلفون والمبلغ. مفيش عنوان الشحن، اسم العميل الضيف، الإيميل، المنتجات المطلوبة، أو الملاحظات.
2. **مفيش إشعارات إيميل**: لا الأدمن بيوصله إيميل لما طلب جديد يتعمل، ولا العميل بيوصله تأكيد.

---

## الحل

### 1. تحسين عرض تفاصيل الطلب في جدول الأوردرات
- تحديث `Order` interface في `useAdminStats.tsx` و `StoreSection.tsx` ليشمل: `guest_name`, `guest_email`, `shipping_address`, `notes`
- تحديث `fetchStats` ليجيب الحقول الجديدة + يجيب `order_items` مع بيانات المنتج
- إضافة **زر "عرض التفاصيل"** لكل طلب يفتح Dialog فيه:
  - اسم العميل (من profile أو guest_name)
  - الإيميل (من auth أو guest_email)
  - التلفون
  - عنوان الشحن كامل
  - قائمة المنتجات المطلوبة (اسم + كمية + سعر)
  - الملاحظات
  - إيصال الدفع (لو موجود)
  - حالة الطلب مع إمكانية التغيير

- تحسين عمود "العميل" في الجدول ليعرض الاسم الحقيقي (guest_name لو ضيف)

### 2. إشعارات الإيميل (مرحلة لاحقة)
إعداد الإيميل يحتاج تفعيل Email Domain أولاً. هنركز دلوقتي على **إصلاح عرض البيانات** وبعدها نضيف الإيميلات.

---

## التفاصيل التقنية

### الملفات المتأثرة

| ملف | تعديل |
|------|--------|
| `src/hooks/useAdminStats.tsx` | إضافة `guest_name`, `guest_email`, `shipping_address`, `notes` للـ Order interface + fetch order_items |
| `src/components/admin/sections/StoreSection.tsx` | تحديث Order interface + إضافة Dialog تفاصيل الطلب مع المنتجات والعنوان |

### تفاصيل الـ Dialog الجديد
- يظهر عند الضغط على زر "عرض" بجانب كل طلب
- يعرض كارت فيه كل بيانات الطلب مقسمة لأقسام:
  - **بيانات العميل**: الاسم، الإيميل، التلفون
  - **عنوان الشحن**: كامل
  - **المنتجات**: جدول صغير (اسم المنتج، الكمية، السعر)
  - **الملاحظات وطريقة الدفع**
  - **الإيصال** (لو موجود)
